# WPS Photos - 图片查看工具

基于 Next.js + Tailwind CSS 的完整图片查看器，使用 react-zoom-pan-pinch 实现专业级缩放拖拽功能。

## 安装依赖

```bash
pnpm install
```

## 启动开发服务器

```bash
pnpm dev
```

浏览器访问 http://localhost:3000

## 🖥️ 安装为桌面应用（PWA）

### 方式一：Chrome/Edge 安装
1. 访问 http://localhost:3000
2. 点击地址栏右侧的 ⊕ 安装图标
3. 点击"安装"按钮
4. 应用将作为独立窗口安装到桌面

### 方式二：手动添加
1. 打开浏览器菜单
2. 选择"将网站安装为应用"或"安装 WPS Photos"
3. 确认安装

### PWA 优势
- ✅ 独立窗口运行，无浏览器界面
- ✅ 离线缓存，加载更快
- ✅ 桌面快捷方式，一键启动
- ✅ 自动更新
- ✅ 原生应用体验

## 功能特性

### 图片浏览
- ✅ 上一张/下一张切换
- ✅ 图片页码显示

### 缩放功能
- ✅ 鼠标滚轮缩放
- ✅ 按钮缩放（+/-）
- ✅ 双击重置
- ✅ 实时显示缩放比例
- ✅ 缩放范围：50% - 500%

### 拖拽功能
- ✅ 缩放后可拖拽移动
- ✅ 平滑拖拽体验

### 旋转功能
- ✅ 左旋转/右旋转（90度）
- ✅ 保持旋转状态

### 快捷键
- `←/→` 上一张/下一张
- `+/-` 放大/缩小
- `0` 重置视图

### 其他
- ✅ 响应式布局
- ✅ 触摸手势支持（移动端）

## 技术栈

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS v3
- Zustand (状态管理)
- react-zoom-pan-pinch (缩放拖拽)
